# Student_Task_Manager
The Student Task Manager is a Single Page Application developed using ReactJS to help students manage their daily academic tasks efficiently. The project uses React components, React Router, and event handling to create a smooth user experience. It includes multiple pages such as Home, Add Task, Task List, and About, implemented using routing without reloading the page. Users can add tasks through a form, view them dynamically, and delete completed tasks easily. The application uses the useState hook for managing task data and updating the interface dynamically. A professional and responsive UI was designed using CSS with a 3/4 page layout. Navigation between pages is handled using React Router DOM. The project demonstrates core React concepts such as components, props, state, and event handling. This mini project is suitable for academic submissions and learning React fundamentals. Overall, the Student Task Manager provides a simple, clean, and efficient productivity tool for students.
